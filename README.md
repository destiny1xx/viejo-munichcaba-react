# Viejo Munich CABA

Sitio web comercial para Viejo Munich CABA, pensado como una landing page moderna para mostrar variedades de cerveza artesanal, packs y recibir pedidos por WhatsApp.

La idea del proyecto es ofrecer una experiencia simple: el usuario puede recorrer las cervezas disponibles, agregar productos o packs a un carrito y finalizar el pedido directamente por WhatsApp, sin necesidad de un sistema de pago complejo.

## Vista general

El sitio incluye:

- presentación principal de la marca;
- catálogo de cervezas;
- sección de pack destacado;
- carrito de pedidos;
- mensaje automático para WhatsApp;
- navegación por secciones;
- diseño responsive para celular y escritorio;
- botón flotante de contacto;
- aviso de venta exclusiva para mayores de 18 años.

## Tecnologías utilizadas

- React
- Vite
- JavaScript
- Tailwind CSS

## Funcionalidades principales

### Catálogo de productos

Cada cerveza cuenta con su imagen, nombre, descripción y perfil. El usuario puede agregar productos al pedido sin salir de la página.

### Carrito por WhatsApp

El carrito permite sumar o quitar productos y luego generar un mensaje automático con el pedido armado. El objetivo es facilitar la compra sin convertir el sitio en un e-commerce completo.

### Pack destacado

La página incluye una sección especial para presentar un pack pensado para regalar o compartir, con una estética más cuidada y llamada a la acción directa.

### Diseño responsive

El diseño está adaptado para verse correctamente tanto en escritorio como en celulares, priorizando una navegación clara y botones fáciles de usar.

## Estado del proyecto

Proyecto en desarrollo, con una primera versión funcional lista para mostrar, probar y ajustar según necesidades reales del negocio.
